## 💻 Sobre o projeto

**BaratieFit** é um site de venda de marmitas saudáveis, desenvolvido para praticar e demonstrar conhecimentos em HTML5, CSS3 e JavaScript.

O desafio proposto foi a criação de uma página moderna, responsiva e voltada para o comércio de marmitas fitness, aplicando boas práticas de desenvolvimento web. O projeto utiliza elementos semânticos do HTML para estruturar o conteúdo de forma organizada, CSS para estilização e responsividade, e JavaScript para interatividade.

> 🚧 Projeto em desenvolvimento — novas funcionalidades sendo adicionadas!
